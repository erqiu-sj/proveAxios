# proveAxios

## install

`npm i @zealforchange/proveaxios`

or

`yarn add @zealforchange/proveaxios`

## why is there this?

```
我们都看腻了，也写腻了各种 axios 请求配置!

在这之前我总是从另一个项目中的 axios 配置拷贝到另一个项目中

这种机械操作让我感到厌倦，我无时不在想着该如何破局

直到他的出现 --- proveAxios
```
